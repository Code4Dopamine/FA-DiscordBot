### Installation & Process:
- Install Node.js (Currently using v18)
- Install Dependencies `npm install`
- Install nodemon globally `npm i nodemon -g`
- Register commands for your discord bot to run, ex: `nodemon src/register-commands.js`
- Run discord bot server (locally) `nodemon`
